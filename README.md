# eCommerce API
## Features
- User CRUD
- Auth
- Product CRUD
- Add and Remove from cart
- Make sales

## Documentation
### To see the endpoints go to this link https://documenter.getpostman.com/view/19464642/UVsHV8Ei

## Server

### Composer Dependencies
To install the dependencies run composer install

### Node Dependencies
To install node dependencies run npm install

### Database
You need to configure a MySQL database to run the API

### Run the server
Use the command php artisan serve to start the server
